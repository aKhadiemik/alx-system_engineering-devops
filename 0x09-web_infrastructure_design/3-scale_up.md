# Scaled Up Web Infrastructure

## Description

Web stack includes distribution and essentially a fault tolerant set up.

## Specifics About This Infrastructure

+ The addition of a firewall between each server.<br/>Distinctly protects each server from unwanted traffic and unauthorized users. 

## Issues With This Infrastructure

+ High maintenance costs.<br/>Operational costs - utility bills, IT skilling, equipment/provisioned cloud resources will require additional budgetary resources.
